# image-tools

一个处理图片的在线工具

![](https://www.gausszhou.top/static/data/github/image-tools/1.webp)

## Features

- [x] 图片压缩
  - [x] 格式转换
  - [x] 单个图片
  - [ ] 多个图片/文件夹
- [x] 图片放缩
  - [x] 格式转换
- [ ] 图片裁剪
  - [ ] 基本裁切
  - [ ] 背景填充
- [x] 去除背景
  - [x] 格式转换
- [ ] 组合工具
